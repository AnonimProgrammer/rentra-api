# Rentra API

Rentra API is the backend service for a car-rental platform where customers discover nearby rental agencies, reserve vehicles, and complete rentals with pricing and post-rent lifecycle handling.
The platform is designed to support practical real-world constraints such as active-rent limits, post-rent technical readiness checks, and role-based operations.
